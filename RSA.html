RSA finding d

The original formula will be "ed mod z = 1"

The formula to find d will be "(zn+1)/e = d"

Where n = 1, 2, 3, 4......

Increment n until d is an integer


For example:

35d mod 264 = 1

The formula to find "d" will be: (264n+1)/35 = d

Where n = 1, 2, 3, 4......

Increment n until d is an integer, which in this case, n is 11 and d is 83
